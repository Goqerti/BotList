# Server'ə quraşdır
```
config.js faylının içərisinə tokeninizi yapıştırın daha sonra👇
git clone https://github.com/goqerti/BotList
cd BotList
npm start
npm install --save telegraf
node bot.js
```

# Heroku'ya deploy et
Əvvəlcə config.js faylının içərisinə bot token yerləşdirin🤗
